Create navigational links within a website

There are several techniques to navigate within a website. The semantic tag for navigation is the <nav> tag. You can navigate through your web site by using relative addressing. For example we will user the unordered list tag. (<ul>)

  • Page to page within the same folder.
  • Page to page between folders.
  • Staying on the current page.
  • Going to a specific place on the current page (on page navigation)

Navigational menus are placed in your <header> semantic area. They may also be placed in your <footer> area as a sitemap. The nav tag like other semantic tags should have a <hx> tag. HOWEVER this header os optional. If you omit the header tag you will receive an document outliner error: Untitled Section. until you are show how to avoid this error message, it will be acceptable on your document.
